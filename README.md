# Film Eleştirisi ve İnceleme Sitesi

Bu proje, üniversite eğitimim kapsamında hazırladığım bir dönem ödevidir. Web geliştirme dünyasına adım atarken HTML, CSS ve JavaScript becerilerimi pekiştirmek amacıyla geliştirdim.

Projenin Amacı ve Durumu
Bu çalışma profesyonel veya ticari bir ürün değildir. Tamamen eğitim amaçlıdır ve öğrenme sürecimin bir parçasıdır. Kodlarda veya tasarımda, o dönemki bilgi seviyemi yansıtan basit çözümler bulunabilir. Amacım, web geliştirme hakkında az çok olsa da bir bilgi sahibi olmaktır.

Neler Yapıldı?
- Filmlerin listelendiği ana sayfa tasarımı.
- Seçilen filme ait detayların ve puanların göründüğü inceleme sayfaları(IMDB)
- Farklı ekran boyutlarına uyum sağlaması için responsive (duyarlı) tasarım çalışmaları.
- JavaScript kullanılarak yapılan temel DOM manipülasyonları ve kullanıcı geri dönütleri.

- HTML5
- CSS3
- JavaScript

Kurulum
Python kurulu ise terminale python -m http.server yazıp devamında adres çubuğuna localhost:8000 yazarak siteye ulaşabilirsiniz.
